Class: HasherWorker

HasherWorker(fileOrString, hasher)

Coordinates between the main process and a Worker process in order to hash a file using a CryptoJS algorithm.

Constructor

new HasherWorker(fileOrString, hasher)

Create a new HasherWorker object.

Initialize the object with:

  • hasher -- a reference to the parent WorkerHasher
  • worker -- a Worker process created using ConstructedWorker#createConstructedWorker
  • string -- the String to be hashed OR
  • file -- the File or string to be hashed
  • chunk -- the index of the current chunk
  • chunks -- the number of chunks in the file
  • reader -- a FileReader object used to read the file

onWorkerMessage(event)

Handle messages sent by the Worker process.

When a create message is received, execute read to begin reading the file.

When an update message is received, execute next to initiate reading the remaining portions of the file.

When a finish message is received, resolve the Promise returned by the call to hash, including the file hash generated.
