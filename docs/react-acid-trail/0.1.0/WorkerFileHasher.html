Class: WorkerFileHasher

WorkerFileHasher(files, algorithmopt, chunkSizeopt)

WorkerFileHasher is the class of objects that coordinates hashing of files using WebWorker processes.

Each worker is an instance of FileHasherWorker

Constructor

new WorkerFileHasher(files, algorithmopt, chunkSizeopt)

Instantiates a WorkerFileHasher, creates a collection for FileHasherWorker instances and calls #createWorker to create a FileHasherWorker for each file in files

workerOnMessage(event)

To be used by FileHasherWorker:

Function provided to FileHasherWorker's internal WebWorkers to handle messages from the FileHasherWorker object.

Upon create message, creates a new CryptoJS algorithm of the type specified when the FileHasherWorker was instantiated. Sends an acknowledgement message back to the caller.

Upon update message, uses the algorithm's update method to update the hash using the data in the message. Sends an acknowledgement message back to the caller.

Upon 'finish' message, returns the hash generated from the file as a String.
