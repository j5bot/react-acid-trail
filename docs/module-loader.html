Module: loader

ipsum-salt/loader module provides methods for loading single-file ipsum generator modules and exposing them via the window object, allowing for lazy loading of various ipsum generators.

Source:

Members

(static, constant) ipsumSalt

Loader method which provides for loading single-file module scripts from the given URL and exposing module members on the window object.

Optionally call an onLoad callback when the module script is loaded.
