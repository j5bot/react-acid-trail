Class: WorkerHasher

WorkerHasher(stringsopt, filesopt, algorithmopt, chunkSizeopt)

WorkerHasher is the class of objects that coordinates hashing of files using WebWorker processes.

Each worker is an instance of HasherWorker

Constructor

new WorkerHasher(stringsopt, filesopt, algorithmopt, chunkSizeopt)

Instantiates a WorkerHasher, creates a collection for HasherWorker instances and calls #createWorker to create a HasherWorker for each file in files

workerOnMessage(event)

To be used by HasherWorker:

Function provided to HasherWorker's internal WebWorkers to handle messages from the HasherWorker object.

Upon create message, creates a new CryptoJS algorithm of the type specified when the HasherWorker was instantiated. Sends an acknowledgement message back to the caller.

Upon update message, uses the algorithm's update method to update the hash using the data in the message. Sends an acknowledgement message back to the caller.

Upon 'finish' message, returns the hash generated from the file as a String.
